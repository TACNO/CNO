【Bilibili-SGTH】B站直播自动抢辣条

· 2020-06-25 · # javascript

以前看到逆回十六夜在greasyfork.org上发布了一个B站自动抢辣条脚本。然后又偶然间发现了SeaLoong的Bilibili-LRHH。

但十六夜的脚本有直播间限制，而且有些地方我觉得可以优化；SeaLoong已经有段时间没更新过了。于是自己重新做了个修改版，在原先基础上做了些改进，如添加新API和新功能。脚本在github上发布，项目名称是Bilibili-SGTH。
欢迎各位来试试，如有建议可以向我反馈✈️
