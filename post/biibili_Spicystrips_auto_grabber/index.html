B站直播自动抢辣条脚本

· 2020-04-06 · # javascript

昨天看到逆回十六夜在greasyfork.org上发布了一个B站自动抢辣条脚本。

但是这个脚本有直播间限制，而且有些地方我觉得可以优化。于是自己重新做了个修改版，在原先基础上做了些改进，如整合API，并添加了新功能。脚本在greasyfork.org上发布，名字叫B站直播自动抢辣条。
欢迎各位来试试，如有建议可以向我反馈✈️

 最后，再次感谢SeaLoong提供的BilibiliAPI v1.4.6。本脚本的BilibiliAPI_mod为在此基础上的修改版。
